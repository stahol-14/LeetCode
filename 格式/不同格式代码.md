# 不同格式的代码

## 核心代码模式

- 就是力扣上的格式，只需要关心写**代码的核心逻辑**即可

如：

```java
class Solution {
    public int[] twoSum(int[] nums, int target) {

        // 算法处理逻辑
    }
}
```

- 什么都没有，需要自己写完整的代码，包括输入输出，数据的处理。

如：

```java

import java.util.Scanner;

public class Main{
    public static void main(String[] args){
        Scanner sc = new Scanner(System.in);

        // 输入处理
        int n = sc.nextInt();

        // 处理逻辑

        sc.close();
    }
}
```
